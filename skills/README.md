# Skills Directory

This directory is used to store installed skills for the Switchboard CLI tool.

Skills are external modules or capabilities that can be installed and used by the CLI to extend its functionality. When a skill is installed, its files and resources are stored in this directory.

## Usage

Skills can be installed, removed, and managed through the Switchboard CLI commands:
- `switchboard skills install` - Install a new skill
- `switchboard skills list` - List installed skills
- `switchboard skills remove` - Remove an installed skill

## Directory Structure

Each installed skill may have its own subdirectory within this folder, containing:
- Skill metadata and configuration
- Skill implementation files
- Any required assets or resources
